# Evis: Visualization for event-based vision data

This repository contains a few functions in C++ to process event camera data efficiently.
They are wrapped to python using pybind11.

# Installation

1. `./prepare_libs.sh`
1. `pip install .`

## Dependencies

1. `pybind11`
1. `Eigen3`


# Requirements

- Python >=3.8
- CMake >=3.1
